AI Email Generation Setup

To generate personalized emails with AI, you'll need an Anthropic API key.

How to get your API key:

  1. Go to console.anthropic.com
  2. Create an account or sign in
  3. Navigate to "API Keys" section
  4. Click "Create Key" and copy it
  5. Paste it below
💡 Cost Information:
  • ~$0.001 per email (~0.1¢)
  • 50 emails ≈ $0.05
  • Uses Claude 3.5 Haiku (most cost-effective)
  • Daily limit: 50 emails
  • Rate limit: 3 per minute
🔒 Privacy & Security:
  • API key stored locally in Chrome (encrypted)
  • Never shared with Ellyn servers
  • You can revoke it anytime
  • Only used when you click "Generate with AI"
